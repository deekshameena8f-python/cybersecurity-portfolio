# Security Log Analyzer

## Overview

The Security Log Analyzer is a Python-based log analysis tool that processes security logs and extracts useful information for security monitoring and incident investigation.

The project demonstrates fundamental Security Operations Center (SOC) workflows used for log analysis and event monitoring.

---

## Features

* Log file parsing
* Event extraction
* Security event identification
* Report generation
* Structured output
* Incident review support
